fennec installed

The fennec daemon binaries, LaunchDaemon, and Helper LaunchAgent configurations are now in place on this Mac.

Next steps

Personal use: open Terminal and run sudo fennec wizard. The wizard will prompt for your org's install secret, register the daemon, and walk you through the SSO attach flow.

Org-tier (MDM): Your IT admin pushes a Configuration Profile via your MDM tool (Jamf, Intune, Workspace ONE). The .pkg postinstall script detects the profile automatically and brings the daemon online without further prompting. The tray notification will appear on next login asking you to sign in.

Where things live

  • /usr/local/fennec/bin/fennec — CLI wrapper
  • /usr/local/fennec/bin/fennec-hook — Claude Code hook shim
  • /usr/local/fennec/bin/fennec-notifier — Helper LaunchAgent binary
  • /Library/LaunchDaemons/dev.fennec.daemon.plist — system daemon
  • /Library/LaunchAgents/dev.fennec.notifier.plist — per-user helper
  • /var/log/fennec/daemon.log — daemon log (root-readable)

To uninstall: sudo fennec uninstall (or your IT admin removes the MDM profile, triggering the uninstall path).
