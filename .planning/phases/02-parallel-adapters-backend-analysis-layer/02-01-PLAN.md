**H6 ‚Äî harden `POST /v1/events` against browser-page forgery.** Add a `POST /v1/events` branch to `handleRequest` calling a new `handleEventsPost`. This route is reachable by ANY web page the developer visits (a page can `fetch("http://127.0.0.1:<port>/v1/events")`), so it needs guards `/v1/hook` (Go-shim-only) does not. `handleEventsPost` runs these checks IN ORDER, before touching the body:
1. **Origin / Sec-Fetch-Site allowlist.** Reject (403, no emit) if `Sec-Fetch-Site` is present and NOT `none`/`same-origin` (a cross-site fetch is `cross-site`), OR if an `Origin` header is present and not in a tight allowlist (the extension's own origin scheme `chrome-extension://‚Ä¶` is the intended caller; a page on `https://chatgpt.com` is NOT). Requests from the paired extension's service worker carry the extension origin / `Sec-Fetch-Site: none`; a hostile page carries `cross-site`. Document the exact allowlist in a comment.
2. **Require `Content-Type: application/json`.** Reject (415, no emit) otherwise. A simple cross-site `fetch` cannot set a non-simple `Content-Type` without triggering a CORS preflight the bridge never answers with permissive headers ‚Äî so requiring `application/json` forces a preflight a hostile page can't satisfy.
3. **Constant-time token compare.** Replace the `!==` string compare with `crypto.timingSafeEqual` over length-checked Buffers: bail to 401 if the provided header is missing or its byte-length differs from the secret (length check first ‚Äî timingSafeEqual throws on unequal lengths), else `timingSafeEqual(Buffer.from(provided), Buffer.from(secret))`. Log only `remoteAddr`, NEVER the secret.
4. **Max body size.** Cap the buffered body at a sane limit (e.g. 256 KB ‚Äî a single prompt post); if `Content-Length` exceeds the cap OR the streamed bytes exceed it, destroy the request and respond 413 (no emit). This prevents an unbounded-body DoS.
Then, as today: `JSON.parse` in try/catch (400, never echo body), `this.emit("events", parsed)`, respond 202. `/v1/hook` and `/v1/health` stay unchanged (the Go shim path keeps its existing posture).
